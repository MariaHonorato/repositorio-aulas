Inserino Links

Para criar links ou hiperlinks devemos criar ancoras atraves da tag . O principal atributo dessa tag é o href, que cria uma referencia hipertexto.
Pode ser um link pra uma página dentro da mesma página, pode ser um link para outra página, ou até para download.
